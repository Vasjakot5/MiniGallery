# Авторизация

# 1.1 Регистрация
- [] Открыть страницу регистрации
- [] Ввести данные
- [] Убедиться, что пользователь создался и редирект на страницу входа

# 1.2 Вход
- [] Открыть страницу входа
- [] Выбрать пользователя из списка
- [] Ввести пароль
- [] Нажать "Войти"
- [] Убедиться, что произошел редирект на главную страницу
- [] В шапке отображается имя пользователя

# Загрузка изображения

# 1.1 Заполнение формы
- [] Убедиться, что пользователь авторизован
- [] Нажать "Загрузка изображения"
- [] Ввести название изображения (например: "Мой тестовый снимок")
- [] Ввести описание (например: "Описание тестового изображения")
- [] Выбрать категорию из выпадающего списка
- [] Выбрать файл изображения (JPG, PNG или WEBP, размером до 5 МБ)
- [] Убедиться, что после выбора файла появляется превью изображения

# 1.2 Отправка формы
- [] Нажать кнопку "Загрузить"
- [] Увидеть индикатор загрузки
- [] Дождаться сообщения об успешной загрузке
- [] Убедиться, что произошел редирект на главную страницу

# 1.3 Проверка результата
- [] На главной странице найти загруженное изображение
- [] Проверить, что отображается правильное название, описание, категория и автор
- [] Нажать на изображение - открывается страница просмотра
- [] Нажать "Мои изображения" и убедиться, что оно там появилось

# Удаление/Редактирование изображения

# 1.2 Удаление изображения
- [] Убедиться, что пользователь авторизован
- [] Найти любое изображение
- [] Нажать кнопку "Удалить" на карточке изображения
- [] Появится диалог подтверждения: "Вы уверены, что хотите удалить это изображение?"
- [] Нажать "OK"
- [] Убедиться, что изображение исчезло из галереи

# 1.4 Проверка на главной странице
- [] Перейти на главную страницу
- [] Убедиться, что удаленное изображение больше не отображается

# 1.2 Редактирование изображения
- [] Найти любое изображение
- [] Попробовать изменить название, картинку и т.д.
- [] Убедиться, что измения произошли

# 1.5 Проверка удаления админом
- [] Войти как администратор (Admin / Admin)
- [] Попробовать удалить изображение другого пользователя
- [] Убедиться, что админ может удалить любое изображение

# 1.5 Проверка удаления админом
- [] Войти как администратор (Admin / Admin)
- [] Попробовать изменить название, картинку и т.д.
- [] Убедиться, что админ может изменить любое изображение

# Фильтрация

# 1.1 Фильтрация по категории
- [] В блоке фильтров выбрать категорию
- [] Убедиться, что в галерее отображаются только изображения выбранной категории

# 1.2 Фильтрация по поисковому запросу
- [] В блоке фильтров ввести поисковый запрос
- [] Убедиться, что в галерее отображаются изображения по поисковому запросу

# 1.3 Фильтрация по автору
- [] В блоке фильтров выбрать автора
- [] Убедиться, что в галерее отображаются только изображения выбанного автора

# 1.4 Комбинированная фильтрация
- [] Выбрать категорию, ввести поисковый запрос и выбрать автора
- [] Убедиться, что фильтры работают вместе

# 1.5 Сброс фильтра
- [] Выбрать "Все категории"
- [] Убедиться, что снова отображаются все изображения
