Инструкция по запуску проекта Mini Gallery

1. Откройте папку проекта и файл .env.

1.1. Измените параметры подключения к базе данных (имя пользователя, пароль, название БД) или оставьте значения по умолчанию.

2. Откройте PostgreSQL (например, через pgAdmin или командную строку) и создайте базу данных с названием, указанным в .env файле.
   
2.1. Затем выполните миграции из папки migrations или импортируйте БД, которое находится в этом проекте

3. Установите Go (если не установлен) и выполните в терминале команду go run cmd/server/main.go.

4. Откройте браузер и перейдите по адресу http://localhost:8080/web/pages/index.html

5. Проверьте, что все работает: изображения загружаются, фильтруются и удаляются.

Данные для входа в админку:
- Администратор: Логин: AdminMiniGallery, Пароль: AdminMiniGallery
